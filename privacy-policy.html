Privacy Policy for Teisei AI

Last updated: December 14, 2023

Data Collection and Usage

The extension requires an OpenAI API key to function. This key is:

  • Stored securely in your browser's local storage
  • Never transmitted to any server except OpenAI's API
  • Can be deleted at any time by clearing the extension's data

Text Processing

When you use the extension to correct or translate text:

  • The selected text is sent to OpenAI's API for processing
  • No text is stored permanently
  • All communication is done over secure HTTPS connections
  • No user tracking or analytics are implemented

Third-Party Services

We use OpenAI's API for text processing. Please review OpenAI's Privacy Policy for information about how they handle data.
